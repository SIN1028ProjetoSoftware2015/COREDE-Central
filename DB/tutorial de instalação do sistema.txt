=======================

2- Criar um banco de dados e importar o "scriptDB.sql" da pasta DB;

3- Configurar no arquivo "./coredeadm/config/sistema.class.php" os dados do banco de dados;

4- Tentar acessar o sistema, se não apresentar erro de conexão, está funcionando!
